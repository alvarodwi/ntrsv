# ntrsv
[n]ikke [t]racing the stars [r]andom [s]elector [v]?
a rhythm game randomizer for nikke minigame, tracing the stars

---

## about
vibe coded using unused claude tokens
motivation is to create randomizer since game haven't provided one
also to have a nice lane cover instead of usual random window

stlye wise it's just barebone but i'm quite satisfied with the layouting

inspired by:
https://github.com/wowvv0w/djmax-random-selector-v
aka dmsrv (should be obvious where this project name come from)

---

## features
- song randomizer
- difficulty filtering
- tag filtering
- album filtering
- rating range filtering (1–19)
- pop up widget synced with main page (can be used as lane cover)
- songs.json overwrite config if the bundled one isn't up-to-date (in the spirit of self service)

---

## song format
songs are loaded from:
public/songs.json

structure is simple and it's easy to change it around too,
currently listing 35 songs that are available in game
[valid per 2026/05/16]
